One could argue that abstraction is a technique that helps us identify which specific information should be visible, and which information should be hidden. - SOLID - S SRP (Single responsibility principle) La noción de que un objeto solo debería tener una única responsabilidad. - O OCP (Open/closed principle) La noción de que las “entidades de software" deben estar abiertas para su extensión, pero cerradas para su modificación. - L LSP (Liskov substitution principle) La noción de que los “objetos de un programa deberían ser reemplazables por instancias de sus subtipos sin alterar el correcto funcionamiento del programa”. - I ISP (Interface segregation principle) La noción de que “muchas interfaces cliente específicas son mejores que una interfaz de propósito general”. - D DIP (Dependency inversion principle) La noción de que se debe “depender de abstracciones, no depender de implementaciones”. - DRY: besides using methods and subroutines in code, rely on code generators, automatic build systems, and so on. Apply it quite broadly to include database schemas, test plans, build system, even documentation. - Occam's Razor: 14th-century logician and theologian William of Ockham states that the **simplest explanation is usually true**. - KISS: simplicity should be a key goal in design and unnecessary complexity should be avoided --- # Architectural Patterns - Layers - MVC - Pipe and Filter - SOA - Event-Driven Architecture - Session ??? - Layers: grouping of related functionality within an application into distinct layers that are stacked vertically on top of each other. --- # 1989-1991 - 1989: Berners Lee envía una propuesta al CERN para el desarrollo un sistema basado en documentos de Hipertexto que permitiría compartir información a los investigadores del CERN. - 1990: Se abre el uso de Internet para fines comerciales, cuando redes comerciales independientes comenzaron a crecer. - 1990: Berners Lee crea HTTP (Hypertext Tranfer Protocol), URI (Uniform Resource Locator), un cliente, define HTML (Hypertext Markup Language) y un server. - 1991: Lee hace público su cliente y el servidor en Internet. ??? CERN: European Organization for Nuclear Research Sir Timothy John Berners-Lee is an English computer scientist, best known as the inventor of the World Wide Web. He made a proposal for an information management system in March 1989, and he implemented the first successful communication between a Hypertext Transfer Protocol (HTTP) client and server via the Internet sometime around mid-November of that same year. Berners-Lee is the director of the World Wide Web Consortium (W3C), which oversees the Web's continued development. He is also the founder of the World Wide Web Foundation. - HTML: HyperText Markup Language. The markup (formatting) language for the Web. - URI: Uniform Resource Identifier. A kind of “address” that is unique and used to identify to each resource on the Web. It is also commonly called a URL. - HTTP: Hypertext Transfer Protocol.
